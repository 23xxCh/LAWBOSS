# 出海法盾 CrossGuard — 项目需求文档 (PRD)

## 背景

CrossGuard（出海法盾）是一个跨境电商智能合规审查平台，帮助电商卖家检测产品描述是否符合目标市场（EU/US/东南亚）的法规要求。

**市场痛点：**
- 跨境电商面临 EU、美国、东南亚等多国法规合规挑战
- 现有工具多为通用 Amazon 运营工具，合规检测能力弱或缺失
- 专业合规服务（如 Legaloo、ChemWatch）价格高昂，仅适合大型企业
- 中小卖家缺乏简单易用、价格合理的合规检测工具

## 竞品分析

### 竞品 1: Helium 10
- **核心功能：** 产品研究、关键词研究、Listing 优化
- **优点：** 品牌知名度高，用户基数大，支持 Amazon/Walmart/TikTok Shop
- **缺点：** 合规检测是附带功能，非核心能力；价格高（$99-399/月）
- **如何超越：** 专注合规检测，功能更深，价格更合理

### 竞品 2: Jungle Scout
- **核心功能：** 产品数据库、销售分析、关键词追踪
- **优点：** 数据量大，产品发现能力强
- **缺点：** 几乎无合规检测功能；主要面向选品需求
- **如何超越：** 将合规检测作为核心功能，而非附属

### 竞品 3: SellerArmor
- **核心功能：** Amazon 账户保护、政策合规监控、申诉服务
- **优点：** 专注账户保护
- **缺点：** 无自助服务工具；主要做被动保护而非主动检测；定价不透明
- **如何超越：** 提供主动的、实时的产品描述合规检测，价格透明

### 竞品 4: Legaloo
- **核心功能：** EU 产品合规检查、化妆品成分合规
- **优点：** 专注 EU 法规，专业性强
- **缺点：** 仅服务企业客户；主要面向德国/欧盟；用户体验复杂
- **如何超越：** SaaS 化、自助式、中小卖家友好

### 竞品 5: ChemWatch
- **核心功能：** 化学品合规管理、SDS 编制、成分筛查
- **优点：** 专业的化学品合规数据库
- **缺点：** 功能过于复杂，仅适合大型化学品/化妆品制造商；不是面向电商的工具
- **如何超越：** 轻量化、面向电商卖家、即开即用的合规检测

### 竞品 6: AMZ123 (中国市场)
- **核心功能：** Amazon 卖家工具聚合、新闻资讯
- **优点：** 中国卖家市场领袖
- **缺点：** 不是合规检测工具本身；是工具聚合器
- **如何超越：** 原生中文支持 + 多语言合规检测 + AI 语义分析

## 我们的独特优势（竞品做不到的）

1. **AI 语义深度检测** — 现有竞品均为关键词匹配，无 AI 语义理解能力。CrossGuard 的 AI 检测器可发现隐含违规、上下文误报等问题。

2. **两级检测流水线** — 关键词初筛（高召回）+ AI 复筛（高精度），兼顾检测速度和准确性。

3. **多市场统一检测** — EU/US/东南亚（新加坡/泰国/马来西亚）一个平台搞定，无需切换多个工具。

4. **词边界匹配** — 英文使用 `\b` 词边界匹配，避免 "painting" 误报为 "pain" 等子串误报问题。

5. **上下文感知替换** — 只替换被检测为违规的词汇，保留合规语境中的词汇。

6. **数据飞轮机制** — 用户反馈持续优化规则，10 次反馈兑换 50 次检测配额，激励用户参与。

7. **中文界面 + 多语言支持** — 针对中国卖家优化，同时支持英文产品描述检测。

## 核心功能

### 1. 合规检测引擎（已实现）
- 医疗宣称检测（MedicalClaimChecker）
- 绝对化用语检测（AbsoluteTermChecker）
- 虚假广告检测（FalseAdChecker）
- 缺失标签检测（MissingLabelChecker）
- 禁用成分检测（BannedIngredientChecker）
- AI 语义深度复筛（可选，配置 LLM API Key）

### 2. 支持的市场与类别
- **欧盟 (EU)：** 化妆品、电子产品、食品、玩具、纺织品
- **美国 (US)：** 化妆品、电子产品、食品、膳食补充剂
- **东南亚：**
  - 新加坡 (SEA_SG)：化妆品、电子产品、食品、膳食补充剂
  - 泰国 (SEA_TH)：化妆品、电子产品、食品、膳食补充剂
  - 马来西亚 (SEA_MY)：化妆品、电子产品、食品、膳食补充剂

### 3. 用户认证与权限
- JWT + bcrypt 密码哈希
- RBAC 角色：admin / user / viewer
- 默认管理员：admin / crossguard2024

### 4. 报告与导出
- 检测报告存储与查询
- PDF 格式导出（Jinja2 + WeasyPrint）
- 批量检测（最多 100 条）

### 5. 数据飞轮
- 用户反馈（false_positive / false_negative / correct）
- 反馈奖励系统（10 次反馈 = 50 配额）
- 规则优化建议生成

### 6. ERP/平台集成
- API Key 认证（X-API-Key header）
- Webhook 回调
- 支持 店小秘、马帮 等 ERP 系统

## 验收标准

1. **功能完整性：**
   - [ ] 5 种违规类型全部可检测
   - [ ] 覆盖 EU/US/SEA_SG/SEA_TH/SEA_MY 全部市场
   - [ ] 批量检测支持 100 条/次
   - [ ] 报告可导出 PDF
   - [ ] 用户反馈功能可用

2. **代码质量：**
   - [ ] 核心引擎测试覆盖
   - [ ] API 集成测试覆盖
   - [ ] 无高危安全漏洞

3. **用户体验：**
   - [ ] 页面加载时间 < 2s
   - [ ] 检测结果清晰易懂
   - [ ] 移动端适配

4. **竞品超越指标：**
   - [ ] AI 语义检测能力 — 竞品无此功能，必须保持领先
   - [ ] 多市场统一 — 竞品多需要多个工具，我们一个平台搞定
   - [ ] 价格 — 比 Helium 10 ($99/月) 更有竞争力
